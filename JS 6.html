MAPS & SETS

Q1)SET

You are given a string (STR) of length N, consisting of only the lower case English alphabet. Your task is to remove all the duplicate occurrences of characters in the string.

Input:
abcadeecfb

Output:
abcdef 

        let s="abcadeecfb";

        let uni=new Set();
        for(i=0;i < s.length;i++){
          uni.add(s[i]);
        }
        let result="";
        for(let key of uni.keys()){
          result+=key;
        }
        console.log(result);

Q2)MAP

You are given a string (STR) of length N, you have to print the count of all alphabet(using maps).

Input:
abcadeecfb

Output:
a=2 b=2 c=2 d=1 e=2 f=1 

        let s="abcadeecfb";
        let myMap= new Map();
        for(let i=0;i < s.length;i++){
         if(myMap.has(s[i])){
         let past=myMap.get(s[i]);
           myMap.set(s[i],past+1);
         }
         else{
           myMap.set(s[i],1);
         }
       }
       for(let [ch, freq] of myMap.entries()){
         console.log(ch+"="+freq);
       }
